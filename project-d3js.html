Project Summary

This visualization transforms a CSV of economic indicators into an interactive lollipop chart. Users can sort, filter, zoom, and hover for tool-tips, exploring trends without ever reloading the page.

Stack : D3 v7, vanilla JS ES6, SVG, Flexbox.

Technical Highlights

  • 💡 Custom d3.scaleBand & d3.scaleLinear for adaptive sizing.
  • ⚡ Lazy-loaded data & deferred GSAP animations (0.9 s FCP).
  • 📱 Responsive SVG viewBox for perfect mobile rendering.
  • 🧩 Modular architecture: one self-contained chart.js class reusable across dashboards.

🚩 Challenge & Goals

Problem Statement

Raw CSV dashboards were static and hard to explore. Users lacked intuitive controls to drill into economic indicator trends in real time.

Objectives

  • Enable dynamic sorting and filtering without refresh
  • Maintain sub-second load and render times
  • Ensure WCAG AA–compliant color contrast

🗺️ Architecture & Data Flow

  • Data Loader: Fetches CSV via XHR and parses with d3.csv().
  • Chart Module: Encapsulated in chart.js, exposes APIs for sort, filter, and zoom.
  • Animation Layer: GSAP timelines trigger on data joins, synced with requestAnimationFrame.
  • UI Controls: Custom dropdowns and sliders bound via event listeners to chart APIs.
